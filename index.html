Steps to deploy a python flask web app to AWS EC2

  • create a sample python flask web app on your local. Push to Github
  • create an AWS EC2 instance.
  • ssh into the instance ssh -i ec2-user@
  • yum install git. yum install pip. pip install flask
  • git clone the python flask web app into the ec2 instance.
  • python3 main.py
  • open browser and access via the webpage via the ec2 public ip. check that inbound security rules allow the port.

Steps create Ec2 instance using terraform

  • Create Git repo for terraform IAC. Add a .gitignore for .terraform folder and *.tfstate*
  • Clone Git repo. run terraform init
  • Write main.tf to create ec2 instance, using existing key pair. pem file contents are not visible.
  • Run terraform plan,apply,destroy to play with it and verify in aws console.
  • Can ssh into ec2 instance using key pair
  • When pushing to Git, verify .terraform folder and *.tfstate* are not pushed
